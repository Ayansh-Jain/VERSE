# React + Vite

This project is an Instagramâ€‘like website called "Verse". It allows users to sign up, log in, follow/unfollow other users, and upload posts. Each user sees a personalized feed (posts from people they follow plus their own posts) with infinite scrolling.

## Available Scripts

- **dev:** Runs the app in development mode using Vite.
- **build:** Builds the app for production.
- **lint:** Runs ESLint for code quality.
- **preview:** Serves the production build locally.

This project uses:
- React
- Vite
- React Router v7
- TailwindCSS (if you wish to use it; current styles are plain CSS)
- ESLint
